
**AMP:**
[https://ampcode.com/free](https://ampcode.com/free) - free coding agent
    

> Choose between `smart` mode (our paid default) and `free` mode on a per-thread basis. No juggling between plans, just choose what’s right for the task.  
> Uses a mix of OSS, production and pre-release frontier models, with some context window and tool use limits.  
> Available on all accounts. Workspace administrators can decide whether to enable Amp Free for their workspace.  
> Supported by ads from developer tool partners we’ve carefully chosen, displayed unobtrusively within the editor extensions and CLI.

**OpenCode:**
[https://github.com/sst/opencode](https://github.com/sst/opencode)
    

> OpenCode includes two built-in agents you can switch between, you can switch between these using the `Tab` key.
> 
> - **build** - Default, full access agent for development work
>     
> - **plan** - Read-only agent for analysis and code exploration
>     
>     - Denies file edits by default
>         
>     - Asks permission before running bash commands
>         
>     - Ideal for exploring unfamiliar codebases or planning changes
>         
> 
> Also, included is a **general** subagent for complex searches and multistep tasks. This is used internally and can be invoked using `@general` in messages.
