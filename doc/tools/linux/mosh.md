## TL;DR
*   **Byobu:** A text-based window manager and terminal multiplexer enhancing GNU Screen functionality.
*   **Mosh:** A remote terminal application that enables stable connectivity across intermittent links by supporting roaming and speculative echo/editing.
*   All three projects operate under the GNU General Public License (GPLv3).
