# Design

"Design" is loosely-used here — this section is really a grab-bag of **desktop UI utilities, screen-sharing, file managers, and small productivity tools** that don't fit neatly into the OS-specific folders.

## Articles

### Screen sharing & remote desktop
- [[deskreen]] — share your screen to any other device on the same WiFi via a browser.
- [[rustdesk]] — open-source TeamViewer / AnyDesk replacement, written in Rust.

### Text & grammar
- [[harper]] — Automattic's open-source English grammar checker (Grammarly/LanguageTool alternative).
- [[ascii_generator]] — quick web-based ASCII-art text generator.

### Disk & files
- [[czkawka]] — find duplicates, similar images, big files, empty dirs (the "spring cleaning" tool).
- [[lstr]] — fast, minimalist Rust `tree` viewer with an interactive mode.
- [[superfile]] — TUI file manager with multi-panel layout (`spf`).

### Personal productivity
- [[weektodo]] — desktop weekly to-do app.
