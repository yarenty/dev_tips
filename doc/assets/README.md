# Assets — image and SVG references

This directory holds the **inline image and SVG assets** referenced from articles across the vault. Files are organised loosely by topic under `img/`.

This is reference material, not navigable content. The mdBook renderer uses these from the articles that embed them; there's no per-file documentation here. Open the relevant article to see how each asset is used.

## Top-level

- `ai_landscape_202502.png` — AI/ML ecosystem map snapshot (Feb 2025).
- `ml-model-cheatsheet.svg` — classical-ML model picker cheatsheet.
- `tmux.png` — tmux pane/window cheat.

## `img/` subdirectories

- `img/cursor.png`, `img/llms.png`, `img/markdown_cheatsheet.jpeg` — top-level inline references.
- `img/wasm/` — WebAssembly status snapshots.
- `img/www/`, `img/kubernetes/`, `img/trading/`, `img/mindsdb/` — per-section illustrations.
