# Fun / whimsical Rust crates (bats and friends)

**Main link:** <https://crates.io/crates/bats>

## Summary

A landing page for "Rust crates that exist mostly for joy." The historically-pinned link here is [`bats`](https://crates.io/crates/bats) — a small CLI that prints bats (and other ASCII spookies) to your terminal. It is genuinely whimsical software with no production purpose, and that's the point: small Rust projects make for great Halloween-week / hackathon / first-Rust-project material because they are end-to-end, fit in a few hundred lines, and exercise CLI parsing, terminal rendering, and `cargo install` distribution.

## Insight

This entry is intentionally a curated whimsy index rather than a single product page. Other Rust projects in similar spirit:

- [`bats`](https://crates.io/crates/bats) — bats in your terminal.
- [[daktilo]] — typewriter sound effects on keypress (also reviewed in this section).
- [`bonsai-rs`](https://crates.io/crates/bonsai-rs) — animated ASCII bonsai tree (Rust port of `cbonsai`).
- [`shotgun-rs`](https://crates.io/crates/shotgun-rs) — pew pew.
- [`oranda`](https://github.com/axodotdev/oranda) — fun-shaped landing-page generator (less whimsical, more "bring delight to docs").
- [`rust-rain`](https://github.com/cowboy8625/rust-rain) — Matrix-style rain for the terminal.
- The whole [[../../../funny/README|funny]] section of this vault — Brainfuck, sleep-sort, esoteric languages, pranks.

Reach for these when you want a small, finishable Rust project to learn from, when you want something silly to break the ice in a terminal demo, or when you're explaining "what's the smallest thing I can publish to crates.io?" to someone new to Rust.
