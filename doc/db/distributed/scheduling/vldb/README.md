# VLDB scheduling papers

A curated folder of **VLDB conference** papers on scheduling — query
scheduling, multi-query optimization, distributed-DB workload management,
and related topics. Authors are from across institutions (this folder is
filed by venue rather than by lab, unlike the `google/` and `microsoft/`
siblings).

The filenames are VLDB proceedings page-numbers (`p<page>-<firstauthor>.pdf`),
which makes it easy to look them up in the VLDB volume index.
